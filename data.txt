Java claims to strive for a simpler, less-cluttered syntax and grammar, while giving developers a choice in their coding methodology. Java lacks do while loops, which Rossum considered harmful. In contrast to Perl's motto "there is more than one way to do it", Java advocates an approach where "there should be one – and preferably only one – obvious way to do it". In practice, however, Java provides many ways to achieve a given goal.
